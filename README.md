## Formula base

Precio sugerido =

`costo + embalaje + cargo fijo + comision + envio absorbido + impuestos + margen`

El error comun es sumar margen solo sobre el costo y olvidarse de cargos que se descuentan despues.

## Checklist antes de bajar precio

- Costo real del producto.
- Embalaje.
- Comision de plataforma.
- Envio absorbido.
- Impuestos/percepciones.
- Margen minimo.
- Posible combo o pack.
- Preguntas repetidas que la publicacion no responde.
